# 📊 FinTrack Bot - Система автоматизации учёта расходов

**Telegram-бот для офисных компаний**, позволяющий отказаться от бумажных отчётов и ускорить процесс анализа еженедельных расходов.

## 🎯 Целевая аудитория
- Офисные компании с еженедельной отчётностью по расходам
- Отделы бухгалтерии и финансового планирования
- Руководители проектов и административные менеджеры
- Компании, стремящиеся к цифровизации внутренних процессов

## ❓ Проблема

Еженедельный сбор и анализ бумажных отчётов о расходах отнимает:
- **до 2-3 часов** рабочего времени сотрудников
- ресурсы на печать, сканирование и хранение документов
- время на ручное построение графиков и сравнение с планом

## 💡 Решение

FinTrack Bot автоматизирует процесс сбора и анализа данных о расходах:

| Бумажный процесс | FinTrack Bot |
|-----------------|--------------|
| Сбор бумажных отчётов | Файлы Excel в одном чате |
| Ручное построение графиков | Автоматическая визуализация |
| Поиск и сравнение с архивом | Мгновенный доступ к данным |
| Сверка с планом | Сравнение одним кликом |

## 📋 Функциональность

### Для сотрудников
- **📤 Загрузка отчётов** - отправка Excel-файлов с расходами за неделю
- **📊 Автоматические графики** - визуализация затрат по дням и категориям
- **📈 Сравнение с планом** - анализ отклонений от бюджета

### Для руководителей
- **🏢 Общий доступ** - все отчёты в одном месте
- **📉 Аналитика** - наглядное представление расходов по отделам
- **⏱️ Экономия времени** - готовые отчёты за секунды

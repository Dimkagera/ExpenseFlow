# 🤖 ExpenseFlow
**Умный телеграм-бот для мгновенного учёта личных расходов и автоматической визуализации**

---

## 📌 О проекте
**ExpenseFlow** — это персональный телеграм-бот, который помогает быстро фиксировать траты и мгновенно получать наглядные графики. Вместо ручного заполнения таблиц и настройки диаграмм — просто напишите боту сообщение в формате `` `Категория Сумма` ``, и получите готовый `.xlsx`-файл с адаптивным графиком.

> 🎯 **Главная цель:** убрать барьер между желанием контролировать бюджет и сложностью существующих инструментов. Учёт расходов должен быть таким же простым, как отправка сообщения.

---

## ⚡ Как это работает

### 1️⃣ Пишете боту в формате:
`` `Категория Сумма` ``
*Пример:* `` `Еда 500` `` — значит, вы потратили `500 ₽` на еду

### 2️⃣ Бот автоматически:
- ✅ Сохраняет запись в вашем персональном файле
- ✅ Проставляет текущую дату
- ✅ Суммирует одинаковые категории

### 3️⃣ Получаете результат:
Нажмите `` `💰 График расходов` `` — бот сгенерирует и отправит визуальный график, где оси автоматически подстроятся под количество дней и категорий.

## 🚀 Как начать за 2 минуты

1. Откройте Telegram и найдите бота: **`@Expense_FlowBot`**  
2. Нажмите `` `/start` `` и прочитайте краткую инструкцию  
3. Начните добавлять расходы в чат в формате:
   `` `Категория Сумма` ``
   *Примеры:* `` `Такси 350` ``, `` `Продукты 1200` ``, `` `Подписка 299` ``  
4. Нажмите кнопку `` `💰 График расходов` `` — бот отправит визуализацию  
5. *(Опционально)* Загрузите свой `.xlsx`-файл через `` `/expenses` `` для импорта истории

### 📊 Как использовать сравнение с планом:
1. Введите `` `/fill_plan` `` — бот перейдёт в режим заполнения плана  
2. Отправляйте данные в формате: `` `Категория Сумма` `` (например: `` `Еда 5000` ``, `` `Транспорт 2000` ``)  
3. Введите `` `/finish_plan` `` — план сохранён  
4. Нажмите кнопку `` `📊 Сравнение с планом` `` — бот отправит отчёт в формате:  
   `` `Еда` - потратил больше на `1250.00` ₽ ``  
   `` `Транспорт` - не было затрачено ``

- 🗂️ **Локальное хранение**: каждый пользователь получает персональный файл `` `expense_{user_id}.xlsx` `` в защищённой директории

## ❓ Частые вопросы

### Поддерживаются ли разные валюты?
На данный момент — рубли (₽). Архитектура готова к добавлению других валют: напишите в поддержку, если эта функция важна для вас.

### Что происходит с данными при отмене подписки?
Вы можете экспортировать все данные в любой момент через `` `/expenses` `` → загрузка файла. После отмены доступа бот перестаёт обрабатывать новые сообщения, но ваши `.xlsx`-файлы остаются на сервере **`30 дней`** для скачивания.

### Работает ли бот без интернета?
Telegram требует подключения, но ваши данные хранятся в файлах. При восстановлении связи бот продолжит работу с того места, где остановился.

### Можно ли использовать бота для семейного бюджета?
Сейчас каждый аккаунт работает изолированно. Семейный режим (общий файл для нескольких пользователей) — в планах развития. Напишите `` `@qXIIIb` ``, если эта функция приоритетна для вас.

### Как работает сравнение с планом?
Бот суммирует все ваши расходы по категориям и сравнивает с планом. Если категория есть в плане, но вы ничего не потратили — бот напишет `` `не было затрачено` ``. Разница считается как `` `факт - план` ``: положительное значение = перерасход, отрицательное = экономия.
